# Offline News App (MVVM)

An Android News application built using Kotlin and Jetpack MVVM Architecture. The app fetches live news using REST APIs and supports offline bookmarking using Room Database.

## Features

- Live news fetching
- Search news articles
- Offline bookmarking
- Clean MVVM architecture
- Repository Pattern
- Navigation Component
- RecyclerView with DiffUtil
- REST API integration

## Tech Stack

- Kotlin
- Android Studio
- XML
- MVVM
- Repository Pattern
- Retrofit
- Room Database
- DAO
- RecyclerView
- DiffUtil
- ViewModel
- LiveData
- Navigation Component
- Coroutines
